# DrawWithFriends

A collaborative drawing iOS app built with SwiftUI.

## Overview

DrawWithFriends is an iOS application that allows users to draw on a canvas using touch gestures. The app provides a simple and intuitive interface for creating drawings on your iPhone or iPad.

## Features

- Touch-based drawing with smooth lines
- Clear canvas functionality
- Support for both iPhone and iPad
- Built with SwiftUI for modern iOS development

## Requirements

- iOS 15.0 or later
- Xcode 15.0 or later
- Swift 5.0 or later

## Getting Started

### Opening the Project

1. Clone this repository
2. Open `DrawWithFriends.xcodeproj` in Xcode
3. Select your target device or simulator
4. Build and run the project (Cmd + R)

### Project Structure

```
DrawWithFriends/
├── DrawWithFriends/
│   ├── DrawWithFriendsApp.swift    # App entry point
│   ├── ContentView.swift            # Main drawing interface
│   ├── Assets.xcassets/             # App assets and icons
│   └── Info.plist                   # App configuration
├── DrawWithFriends.xcodeproj/       # Xcode project file
└── README.md                        # This file
```

## How to Use

1. Launch the app
2. Tap and drag on the white canvas to draw
3. Use the "Clear" button to start a new drawing
